# Luminlex - 英语题目生成器

一个简单的英语题目生成工具，点击按钮即可生成四六级等英语考试题目。

## 功能特点

- 🎯 **简单易用**：无需复杂配置，直接选择选项即可生成题目
- 📚 **多种考试类型**：支持CET-4、CET-6、TEM-4、TEM-8、IELTS、TOEFL
- 🔧 **多种题目类型**：听力、阅读、写作、翻译
- 🎨 **暖色调界面**：简洁美观的用户界面
- 📥 **题目导出**：支持JSON格式导出生成的题目
- 📜 **历史记录**：自动保存生成历史，方便回顾

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行应用

```bash
streamlit run streamlit_app.py
```

### 访问应用

打开浏览器访问：`http://localhost:8501`

## 使用说明

1. **选择考试类型**：从下拉菜单中选择需要的考试类型（如CET-4）
2. **选择题目类型**：选择听力、阅读、写作或翻译
3. **设置难度**：选择简单、中等或困难
4. **输入主题**（可选）：指定题目主题，如"环境保护"
5. **生成题目**：点击"生成题目"按钮
6. **查看结果**：查看生成的题目、答案和解析
7. **导出题目**：可下载JSON格式的题目文件

## 技术栈

- **前端**：Streamlit
- **后端**：Python
- **数据格式**：JSON
- **版本控制**：Git

## 注意事项

1. 当前版本使用模拟数据生成题目
2. 所有生成的数据仅保存在浏览器会话中，刷新页面会清空
3. 如需持久化保存，请使用导出功能下载题目文件
4. 敏感配置文件（如secrets.toml）已从git历史中删除并被.gitignore忽略
